## File Interaction
- `find` - Finds a specific file within a tree of directories.
- `grep` - Searches the contents of a file.
- `touch` - Create file.
- `mkdir` - Makes new directory.
- `cp` - Copy files.
- `mv` - Move files.
- `rm` - Remove files.
- `file` - Displays information about the file type.

## File Permissions
- `-r` - Read permission.
- `-w` - Write permission.
- `-x` - Execute permission.
