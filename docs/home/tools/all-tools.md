## Forensic Tools
### File
- Displays basic information about the file song.mp3.
- Usage: `file song.mp3`.

### Exif Tool
- Displays information on file metadata.
- Usage: `exiftool song.mp3`.

### CyberChef
- For encoding and decoding.
- Link: https://gchq.github.io/CyberChef
